Simulador de línea de producción desarrollado en Python.

El objetivo del proyecto es modelar una línea de producción industrial utilizando conceptos de:

    Programación orientada a objetos (POO)
    Estructuras de datos
    Colas y procesamiento de tareas
    Simulación de eventos
    Gestión de estados
    Algoritmos

Objetivos del proyecto

    Este proyecto ha sido creado como práctica para mejorar habilidades de programación y diseño de software.

Características que se implementarán progresivamente:

    Gestión de estaciones de trabajo
    Procesamiento de piezas
    Colas de producción
    Control de tiempos
    Simulación por ciclos
    Estadísticas de producción
    Detección de cuellos de botella
    Exportación de datos
    Interfaz visual futura (opcional)

Tecnologías utilizadas:
    Python 3
    GitHub

Conceptos aplicados:

Programación orientada a objetos:
    Clases
    Encapsulación
    Herencia
    Polimorfismo

Estructuras de datos:
    Queues
    Lists
    Dictionaries
    Graphs (futuro)

Algoritmos:
    Simulación paso a paso
    Gestión de eventos
    Planificación de tareas
